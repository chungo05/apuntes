#### Tipos de datos en oracle
- VARCHAR2(n) - TEXTOS DE LONGITUD n.
- NUMBER - NUMEROS ENTEROS (SIN DECIMALES)
- NUMBER (X,Y) - NUMEROS CON DECIMALES
  X = CANTIDAD TOTAL DE DIGITOS
  Y = CANTIDAD DE VALORES DECIMALES
    Por ejemplo si quiero agregar un "1000.00" seria = NUMBER(6,2)
- DATE - Fechas
