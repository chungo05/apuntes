### Sintaxis

* Identificador tipoDeDato := valor;
**se empieza las variables con el tipo de dato**
ejemplo:
* v_hiredate DATE;
* v_deptno NUMBER(2) NOT NULL := 10;
* v_location VARCHAR(13) : = 'Atlanta';
* c_comn CONSTANT NUMBER := 1400;
* v_valid BOOLEAN NOT NULL := TRUE;

variables con el atributo %TYPE

* v_ename employees.last_name%TYPE;
* v_balance NUMBER(7,2);
* v_min_balance v_balance%TYPE := 10;

Atributo %ROWTYPE: Permite declarar una vairable para almacenar la misma información sobre una fila de una tabla determinada

dept_record departments%ROWTYPE;
emp_record employees%ROWTYPE;
