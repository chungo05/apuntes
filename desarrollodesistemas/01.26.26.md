# CSS Cascading style Sheets

Hojas de estilo en cascada es el lenguaje utilizado para describir la
presentación de documentos.

## Buena practica

El html y el css no debe de estar en el mismo archivo.

un archivo que sea solo HTML y otro archivo que solo sea CSS.
¿Porque?

* Para reutilizar código.
* Si quiero cambiar un botón que se repite, con solo cambiarlo en
 css se cambia en todos lados.

CSS tiene un sintaxis simples en ingles.
A css declaration:
Background-color (property).
: (the colon separates two ientities).
red (the value we are changing.)
